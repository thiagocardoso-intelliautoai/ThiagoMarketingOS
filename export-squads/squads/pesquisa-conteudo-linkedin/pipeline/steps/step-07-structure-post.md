# Step 07: Estruturar Post

## Instrução

Com o hook escolhido pelo usuário, estruturar o post completo seguindo o `post-structure-linkedin.md`.

### Processo:

1. Selecionar o framework mais adequado (PAS, Contraste, Storytelling, Lista, Declaração+Defesa)
2. Informar o framework escolhido e por quê (1 linha)
3. Escrever o post completo no formato:

```
## Post Draft

**Framework:** [nome do framework] — [motivo em 1 frase]
**Pilar ACRE:** [A/C/R/E]
**Chars totais:** [contagem com espaços]

---

[Hook escolhido]

[Espaço]

[Body — parágrafos de max 2 linhas]

[Espaço]

[Insight / Takeaway]

[Espaço]

[CTA]

[hashtags opcionais]

---

### Notas do Redator
- [Observação sobre escolhas feitas ou alternativas]
```

### Checklist de auto-revisão (Redator faz antes de entregar):

- [ ] Hook ≤ 210 chars ✓
- [ ] Post total ≤ 1.300 chars com espaços ✓
- [ ] Parágrafos max 2 linhas ✓
- [ ] Uma ideia central ✓
- [ ] CTA é pergunta ou ação específica ✓
- [ ] Tom de voz consistente (checou `tone-of-voice.md`) ✓
- [ ] Sem vocabulário proibido ✓

## Output
Post rascunho salvo em `output/post-draft.md` → Vai para revisão (step-08)
