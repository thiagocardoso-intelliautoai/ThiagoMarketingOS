# Quality Criteria — Capas LinkedIn & Instagram

## Design — Todos os Estilos

- [ ] HTML self-contained: inline CSS, sem deps externas exceto Google Fonts
- [ ] Viewport exato: 1080 x 1350
- [ ] Texto máximo: 20 palavras na capa
- [ ] Font sizes atendem mínimos (Hero mín. 36px, Caption mín. 18px)
- [ ] Contraste WCAG AA 4.5:1 entre texto e fundo
- [ ] Design system documentado antes da criação
- [ ] Render verificado via browser tool screenshot

## Design — Rascunho no Papel

- [ ] Foto real do banco Supabase (`source_photos`, category=papers) — não gerada por IA. Fallback: `assets/papers/`
- [ ] Infográfico desenhado SOBRE o papel na foto
- [ ] Estilo lápis/caneta — não tipografia digital
- [ ] Cenário ao redor do papel intacto (não editado)
- [ ] Rascunho legível em mobile (texto claro, não rabisco)
- [ ] Máx. 5 blocos de informação no infográfico
- [ ] Marca discreta fora do papel

## Design — Pessoa + Texto

- [ ] Foto do banco Supabase (`source_photos`, category=photos) contextualizada ao tema. Fallback: `assets/photos/`
- [ ] Overlay gradient 60%+ para contraste
- [ ] Texto posicionado na lei dos terços
- [ ] Text-shadow aplicado
- [ ] Se usou IA: resultado realista, face/expressão preservadas

## Design — Micro-Infográfico

- [ ] APENAS UM dado/número principal
- [ ] Fonte do dado citada
- [ ] Número em tamanho hero (72-96px)
- [ ] Legenda clara do que o número significa

## Design — Print de Autoridade

- [ ] Header com foto circular + nome + @ do Thiago
- [ ] Print legível e centralizado
- [ ] Texto do Thiago é o elemento principal (print é suporte)
- [ ] Atribuição da fonte do print

## Design — Quote Card

- [ ] Aspas decorativas como elemento gráfico
- [ ] Separador visual entre citação e atribuição
- [ ] Atribuição presente

## Review (Rui Revisão)

| Critério | Peso | Threshold |
|----------|------|-----------|
| Impacto Visual | Alto | >= 7/10 |
| Legibilidade Mobile | Alto | >= 7/10 |
| Alinhamento Tom de Voz | Médio | >= 6/10 |
| Aderência ACRE | Médio | >= 6/10 |
| Consistência Visual | Alto | >= 7/10 |

**APPROVE:** Média >= 7/10 e nenhum critério abaixo de 5/10
**REJECT:** Média < 7/10 OU qualquer critério abaixo de 5/10
