# Review Checklist — Capas LinkedIn & Instagram

## Revisão Visual Rápida

### Impacto (~2 segundos de avaliação)
- [ ] A capa para o scroll? (teste: olhe por 1 segundo e desvie o olhar — lembrou da mensagem?)
- [ ] Hierarquia visual clara — sabe onde olhar primeiro?
- [ ] Não compete visualmente com texto do post (complementa, não repete)

### Legibilidade
- [ ] Texto legível em tela de smartphone (imagine 375px de largura)
- [ ] Font sizes atendem os mínimos do estilo (ver visual-styles.md)
- [ ] Contraste WCAG AA 4.5:1 entre texto e fundo
- [ ] Se tem overlay sobre foto: opacidade 60%+ garante leitura

### Tom de Voz
- [ ] Consistente com o tom do post que acompanha
- [ ] Sem termos proibidos (ver linkedin-strategy.md)
- [ ] Anti-guru: sem hype, sem clichê, sem emoji emocional
- [ ] Texto falsificável ou opinativo (não genérico)

### Aderência Estratégica
- [ ] Serve ao pilar ACRE correto do post
- [ ] Formato adequado à plataforma (LinkedIn + Instagram)
- [ ] Viewport correto: 1080 x 1350

### Técnico
- [ ] HTML self-contained (zero CDN exceto Google Fonts)
- [ ] Viewport exato no body: width:1080px height:1350px
- [ ] PNG renderizado sem artefatos visuais
- [ ] Sem placeholder text em nenhum elemento
- [ ] Design system documentado
