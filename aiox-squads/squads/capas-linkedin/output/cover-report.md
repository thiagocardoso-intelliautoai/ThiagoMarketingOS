## Photo Selection
- **Selected:** `IMG_20260225_201815722.jpg`
- **Context:** Selfie com equipe num bar — Thiago + 2 colegas com camiseta Winning Sales, thumbs up
- **Adaptation:** Não (foto original sem alteração por IA)
- **Rationale:** A foto é autêntica, mostra a equipe reunida num momento informal, e um dos colegas usa camiseta Winning Sales. Perfeito para post sobre cultura de equipe e IA — transmite proximidade, bastidor e energia de time.

## Design Rationale
1. **Texto alinhado à esquerda** (terço inferior) — assimetria intencional, anti-simetria de IA
2. **Overlay bottom-heavy** (92% na base) — foto visível no topo, texto protegido na base
3. **"cultura" em Teal** — destaque semântico na palavra-chave do post (1 accent, flat)
4. **Zero efeitos** — sem glow, sem gradient text, sem ícones, sem boxes decorativos
5. **Espaço negativo 40%+** — a foto domina e respira, o texto é conciso
6. **Referência editorial** — layout que parece report cover ou slide de bastidor

## Anti-AI Compliance Checklist
- [x] Zero gradient text fills
- [x] Zero radial glows / background glows
- [x] Zero ícones literais/óbvios
- [x] Não é 100% centralizado — assimetria intencional (texto esquerda, terço inferior)
- [x] Máx. 2 elementos decorativos (accent-line + brand)
- [x] Cor de acento é flat (Teal #14B8A6)
- [x] 30%+ espaço negativo (foto respira no topo)
- [x] Paleta não é "óbvia"
- [x] Cores usadas estão no Design System global
- [x] Passa o "teste do olho": parece editorial, não template
