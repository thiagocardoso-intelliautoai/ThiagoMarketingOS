## Objetivo

Aprofundar os insights selecionados no step-02 com dados adicionais,
fontes verificáveis e contexto do mercado brasileiro.

---

## Instruções

### 1. Receber Insights Selecionados
- Ler os insights que o Thiago aprovou no checkpoint step-02
- Nota sobre contexto ou prioridade que ele adicionou

### 2. Aprofundar Cada Insight
Para cada insight selecionado:
- Buscar dados complementares nas fontes Tier 1-3
- Adicionar pelo menos 1 fonte adicional de suporte
- Contextualizar para o mercado brasileiro B2B
- Identificar contra-argumentos (se houver)

### 3. Preparar para Armazém
Para cada insight aprofundado, preparar no formato:

```
## 💡 [Título da Ideia]

- **Tema:** [tema central]
- **Pilar ACRE:** [A/C/R/E]
- **Insight central:** [a ideia em 1-2 frases]
- **Dados de suporte:**
  - [Dado 1 — fonte + data]
  - [Dado 2 — fonte + data]
- **Ângulo sugerido:** [como transformar em post]
- **Contexto BR:** [como isso se aplica no Brasil]
- **Status:** 🟡 Pendente seleção
```
