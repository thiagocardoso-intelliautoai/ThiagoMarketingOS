# Task: Pesquisa Semanal

> Step 01 do pipeline — Pesquisador varre fontes e identifica insights.

---

## Objetivo

Pesquisar fontes Tier 1/2/3 e concorrentes para identificar 5-10 insights
relevantes para o ICP (Carlos Oliveira — Gerente/Diretor de Vendas B2B).

---

## Instruções

### 1. Varrer Fontes Tier 1 + Tier 2
- Consultar `data/research-sources.md` para lista completa
- Priorizar: Gartner, Forrester, McKinsey, HubSpot, Salesforce, LinkedIn Economic Graph
- Secundário: GTMnow, SaaStr, Gong Labs, Sales Hacker

### 2. Benchmark de Concorrentes
- Consultar `data/competitors.md` para lista de concorrentes
- Analisar posts recentes dos Tier 1 (weekly)
- Extrair frameworks e ângulos (NUNCA copiar copy)

### 3. Classificar Insights
Para cada insight identificado, preencher:

```
### Insight #[N] — "[Título curto]"
- **Fonte:** [nome + data]
- **Dado-chave:** [número, estatística ou informação principal]
- **Por que importa pro ICP:** [1 frase]
- **Ângulo para post:** [1-2 frases]
- **Pilar ACRE sugerido:** [A/C/R/E + justificativa]
- **Contexto BR:** [como isso se aplica no mercado brasileiro B2B]
- **Contra-argumento:** [visão oposta, se houver — 1 frase]
- **Urgência:** 🔴 Urgente | 🟡 Relevante | 🟢 Pode esperar
```

### 4. Filtrar para 3-5 Melhores
Critérios de filtragem:
- ✅ Tem dado verificável
- ✅ Relevante pro ICP
- ✅ Tem potencial de post (ângulo claro)
- ❌ Não é hype sem dado
- ❌ Não é genérico
