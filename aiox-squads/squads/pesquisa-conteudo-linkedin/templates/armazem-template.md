# Template: Entrada no Armazém de Ideias

> Template para registrar ideias aprovadas que ainda não foram produzidas.

---

```
## 💡 [Título da Ideia] — [Data de Registro]

- **Tema:** [tema central]
- **Pilar ACRE:** [A/C/R/E]
- **Insight central:** [a ideia em 1-2 frases]
- **Dados de suporte:**
  - [Dado 1 — fonte + data]
  - [Dado 2 — fonte + data]
- **Ângulo sugerido:** [como transformar em post]
- **Contexto BR:** [como isso se aplica no Brasil]
- **Urgência:** 🔴 Urgente | 🟡 Relevante | 🟢 Pode esperar
- **Status:** 🟡 No armazém | 🔵 Em produção | ✅ Publicado

---
```

## Regras do Armazém

1. **Sempre registrar com data** — Para acompanhar validade do insight
2. **Revisar mensalmente** — Insights têm vida útil; remover os que perderam relevância
3. **Max 20 ideias ativas** — Se passar, é hora de produzir ou descartar
4. **Status atualizado** — Mover para "Em produção" quando iniciado, "Publicado" quando saiu
