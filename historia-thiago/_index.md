## 📝 Formato das entradas nos arquivos temáticos

Toda história em `jornada-profissional.md` e `evolucao-pessoal.md` segue:

```
## [YYYY-MM] Título

### Cena
(contexto físico/temporal — onde, quando, quem está)

### Como ele conta (fala literal)
> citações diretas do Thiago, preservando "brother", "leque", "caralho"

### O concreto
- nomes, números, datas, métricas

### Lições / insights extraíveis
- bullet points com aprendizados

### Use cases
- post sobre X, post sobre Y, hook: "..."

### Tags semânticas
#tema #subtema #pessoa #ano-mes
```

Isso é o formato esperado de retorno do subagent — preserva a voz do Thiago para storytelling autêntico.
